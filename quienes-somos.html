Quiénes Somos

Somos el Grupo LCR, encargado del desarrollo del proyecto Sistema de Gestión de Inventario para la materia Full Stack I.

Este sistema fue creado con el objetivo de mejorar la organización, administración y control del stock de productos dentro de una empresa.

Objetivo del Sistema

El sistema permite registrar productos, controlar entradas y salidas de stock y mantener la información actualizada para facilitar la toma de decisiones y reducir errores en la gestión de inventario.

Información del Proyecto

Proyecto académico desarrollado para practicar conceptos de desarrollo web y gestión de sistemas de inventario.
