# Sistema de Gestión de Inventario - Grupo LCR

¡Bienvenido al repositorio del **Sistema de Gestión de Inventario**! Esta aplicación web está diseñada para administrar y controlar de manera eficiente los productos y recursos físicos de una organización. El sistema cubre desde el registro de nuevos usuarios y portales de acceso seguros, hasta el control de stock en tiempo real con formularios dinámicos y módulos de auditoría analítica con alertas de existencias.

Proyecto académico desarrollado para la materia **Full Stack I (2026)**.

## Instrucciones de Instalación y Ejecución Local

*   Clona este repositorio en tu espacio de trabajo local

*   Mantén la integridad de las carpetas para asegurar que las referencias relativas cruzadas hacia los estilos compartidos (`../frontend/css/global.css`) resuelvan en el navegador sin problemas desde la carpeta de vistas.

*   Abre cualquiera de los archivos HTML (se recomienda iniciar desde `index.html` o `login.html`) en tu navegador web de preferencia para interactuar con la interfaz estática de la aplicación.
